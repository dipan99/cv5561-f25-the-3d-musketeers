# 3D Room Reconstruction and Interactive Object Manipulation

## Quick Start

**New to this project?** See [SETUP.md](SETUP.md) for complete installation and setup instructions using Pipenv.

```bash
# Install dependencies
pipenv install

# Activate environment
pipenv shell

# Run SfM preparation
python prepare_for_3dgs.py
```
